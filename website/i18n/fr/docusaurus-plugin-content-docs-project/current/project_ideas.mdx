# Idées de Projets

Avant de choisir ce que vous souhaitez réaliser, il peut être utile de réfléchir aux questions suivantes :

- Trouvez-vous ce projet **utile** ou **amusant** ?
- Quel est le **niveau de complexité** de votre projet ? Combien d’heures pensez-vous y consacrer ? (Il devrait représenter environ **40 heures** de travail)
- Quelles compétences pouvez-vous **acquérir** en réalisant ce projet ?
